*java.sql.SQLTimeoutException_Description*

The subclass of (|java.sql.SQLException|) thrown when the timeout specified by Statement has expired. This exception does not 
correspond to a standard SQLState. 



*java.sql.SQLTimeoutException()*

public SQLTimeoutException()

Constructs a SQLTimeoutException object. The reason, SQLState are initialized to null and the vendor code is initialized to 0. 

The cause is not initialized, and may subsequently be initialized by a call to the (|java.lang.Throwable|) method.
